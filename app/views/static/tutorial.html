Curious as to how this new-fangled Labyrinth Forum works?

Then you're in luck! This tutorial will guide you through signing up and answering questions

  1. You'll already have been registered with Labyrinth following our latest release, so all you have to do is start writing!
  2. Go to the submissions page and just start typing the name of whoever you'd like to send your question to, and if they've signed up for Labyrinth, it'll appear in the dropdown box. If you have any questions or concerns, email us at labyrinth@boltoncomputing.com
