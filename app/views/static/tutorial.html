Curious as to how this new-fangled Labyrinth Forum works?

Then you're in luck! This tutorial will guide you through signing up and answering questions

  1. The first step is to make sure that you're registered for Labyrinth's New Forum by registering at the following link
  2. Registration Link
  3. Fill in the form with your full name as you'd like to see it appear in Labyrinth, and make sure you select the proper grade, or we won't be able to verify you as a student
  4. That's it! If everything happens properly, you'll have been added as a user and be redirected to the Question Submission Page. Just start typing the name of whoever you'd like to send your question to, and if they've signed up for Labyrinth, it'll appear in the dropdown box. If you have any questions or concerns, please email us at labyrinth@boltoncomputing.com
